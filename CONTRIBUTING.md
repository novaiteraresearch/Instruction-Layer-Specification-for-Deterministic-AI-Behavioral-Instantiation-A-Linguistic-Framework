# Contributing Guidelines

## Repository Purpose

This repository documents and provides citation metadata for the proprietary technical publication:

**Instruction-Layer Specification for Deterministic AI Behavioral Instantiation: A Linguistic Framework for Computational Agency Control**
Ariel J. Furlow, Nova Itera Research Group (2026)
DOI: https://doi.org/10.5281/zenodo.18786639

This repository does not contain executable source code and does not accept general public contributions to its documentation or specification content.

You are welcome to use the materials in this repository for evaluation, citation, and assessment purposes, within the limits of the proprietary LICENSE. No special action is required.

---

## What Is Not Accepted

- Unsolicited pull requests modifying documentation, specifications, or citation metadata
- Issues requesting disclosure of proprietary implementation details
- Requests to relicense this work or any portion thereof

Unsolicited pull requests will be closed without review to preserve the integrity and provenance of the underlying research artifact.

---

## Reporting Errors in Citation Metadata

If you identify a factual error in the citation metadata (CITATION.cff, README.md) such as an incorrect DOI, ORCID, or date, you may open an issue describing the discrepancy. Issues of this nature will be reviewed by the maintainer.
